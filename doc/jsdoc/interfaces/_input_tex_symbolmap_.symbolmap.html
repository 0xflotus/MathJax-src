Interface SymbolMap

SymbolMaps are the base components for the input parsers.

They provide a contains method that checks if a map is applicable (contains) a particular string. Implementing classes then perform the actual symbol parsing, from simple regular expression test, straight forward symbol mapping to transformational functionality on the parsed string.
